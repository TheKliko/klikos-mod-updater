Kliko's modloader

A Roblox modloader inspired by Bloxstrap. It serves as a wrapper around Roblox's default launcher, allowing users to easily launch a modded copy without any risk of getting their account suspended. Python 3.12 and a PC running Windows are required.

Here are some if its key features:

  • Mods
  • FastFlags
  • Discord RPC
  • Studio support
  • Launching from web
